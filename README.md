# Professional CV - Luciano N. Navarrete Binni

This is my professional Curriculum Vitae website, designed with a clean, minimalist aesthetic and interactive features.

## Features

-   **Professional Design**: Clean layout with "Slate & Sky" color palette.
-   **Interactive Controls**:
    -   **Dark/Light Mode**: Toggle between themes.
    -   **Language Switch**: English (Default) and Spanish support.
    -   **PDF Download**: Native print-to-PDF integration.
-   **Responsive**: Optimized for Desktop and Mobile.
-   **ATS Friendly**: Semantic HTML structure.

## Structure

-   `index.html`: Main CV structure.
-   `css/styles.css`: Custom styles with CSS variables for theming.
-   `js/script.js`: Logic for theme, language, and PDF features.
-   `js/translations.js`: Content strings for English and Spanish.
-   `img/`: Profile picture and assets.

## Technologies

-   HTML5
-   CSS3 (Variables, Flexbox)
-   JavaScript (ES6+)
-   Font Awesome (Icons)
-   Google Fonts (Inter)
